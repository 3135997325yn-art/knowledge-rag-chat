Local RAG Knowledge Base Sample Document

This project is a retrieval-augmented generation (RAG) system built with:
- Python
- LangChain for document processing and chains
- FAISS for local vector storage
- sentence-transformers for semantic search
- Ollama (llama3) for answer generation

Workflow:
1. Upload documents through the API or CLI.
2. Split text into chunks with configurable size and overlap.
3. Embed chunks and store vectors in a persistent FAISS index.
4. Retrieve the most relevant chunks for a user question.
5. Pass retrieved context to Ollama llama3 to generate grounded answers.

Supported file formats include TXT, Markdown, PDF, DOCX, and CSV.
